# Checklist de Mudança (Obrigatório)

Antes de qualquer alteração de código, confirme:

- [ ] Revisei ADR-001 (Transcrição imutável)
- [ ] Revisei ADR-002 (Estrutura de Ata / Resumo)
- [ ] Revisei ADR-003 (Chunking semântico)
- [ ] Verifiquei se a mudança viola alguma decisão existente
- [ ] Caso viole, registrei nova decisão em DECISIONS.md
- [ ] A alteração é mínima e localizada
