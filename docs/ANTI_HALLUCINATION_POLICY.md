# ANTI_HALLUCINATION_POLICY.md

Política anti-alucinação do projeto.

---

## Regras

- Toda afirmação deve estar no texto fonte
- Se não estiver, responder: "não consta"
- Nenhuma inferência é permitida
- Nenhuma criatividade é desejada

---

## RAG

Permitido apenas para:
- Documentos institucionais
- Atas anteriores

Proibido para:
- Completar reunião atual
- Corrigir transcrição
